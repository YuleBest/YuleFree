我们支持的设备

满足以下全部条件的可以体验当前版本的 Shell 工具箱

  • 支持基本的 Shell 环境

  • 使用 ARM 64 架构的处理器（后续会适配更多）

    你可以在终端执行 uname -m 来获取你的处理器架构，若返回值为 aarch64 即为 ARM64 架构

  • 拥有 SU 权限 (Root)

快速上手

  1. 在本项目的 Releases 页面获取 Shell 工具箱的发行版，你会得到一个 .zip 压缩包

  2. 将压缩包内的所有文件解压到同一目录

  3. 运行 Shell工具箱-R-X.sh

    我们推荐你使用 MT 管理器的「系统环境」进行，需要授权 Root

    初次运行时，系统会提示是否允许在固定目录下生成必要文件，若不允许，Shell 工具箱将无法运行

当前功能列表（R-1）

一. 系统优化

ID 标识号 加入的版本 作者 功能
1 XFZT R-1 Yule 修复字体模块于 Android 15 系统上失效
2 PLAZ R-1 Yule 批量安装指定目录下的 APK 安装包
3 ZCZY R-1 Yule 找出占用空间达到某个值的文件
4 TQSJ R-1 Yule 提取手机系统分区镜像文件
5 BFZK R-1 Yule 备份字库 (可自定义排除)
